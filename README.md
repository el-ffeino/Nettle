# Nettle
Live wallpapers on any Linux running GNOME desktop environment with easy-to-use graphical user interface.  
Written in C# using GTK/Libadwaita as the UI backend with minimalism in mind.  

Comes with a complementary GNOME extension for additional shell patches.  
With **Nettle** you can even set your live wallpaper as a workspace preview actor, as well as the overview background itself.  
Supports multiple display setups!

# Features
- Set any video as your desktop background
- User-friendly graphical user interface
- Drag 'n' drop videos to import them
- Start with system startup
- Complete multi-monitor support
- Play different wallpapers on different monitors
- Automatically pause playback when another app is maximized/fullscreen
- Replace workspace preview actor with the wallpaper itself
- Doesn't consume much of your system resources*
- Shell patches also support multiple display setups*

**And more!**

*1) Around 2.5% CPU 3.5% GPU usage for each instance - depending on your hardware.*  
*2) Both shell patches are experimental and may lead to issues. If you experience problems, you can disable them in settings.*

# Installation
Download any of the [releases](https://github.com/el-ffeino/Nettle/releases) and extract into your desired location.  
From there on, you can just run `Nettle` executable.  

### Dependencies
Latest version of GNOME, mpv, dotnet runtime  

```console
sudo pacman -S ffmpeg dotnet-runtime
```

### Known bug(s)
Sometimes wallpaper might fail to attach onto desktop, either leave it running and it'll fix itself or just quit and play it again.  
*You can find a list of potential new features / bug fixes inside the \`To Do\` file.*
